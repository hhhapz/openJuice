< TUTORIAL_00 >
100% Orange Juiceへようこそ！\nここではゲームの基本を説明します。\n不要な場合はスキップできます。

< TUTORIAL_01 >
このゲームは4人用ボードゲームです。\nノルマ(目的)を達成し、最初にノルマ\nレベルを6にしたプレイヤーが勝ちです。

< TUTORIAL_02 >
ダイスを振り、ボード上のマス\nを移動します。\nマスには色があり、\n異なる効果を持っています。

< TUTORIAL_03 >
これはホームマスです。\nゲーム開始時にプレイヤーは必ずこのマスにいます。\nまた、条件を満たしてこのマスにとまるとノルマレベルが上がります。

< TUTORIAL_04 >
各プレイヤーは自身の番号が付いたホームマスを持っており、ダイスで丁度の目が出なくても、そのマスに止まるかどうかを選択できます。\n他のプレイヤーのホームマスにも、丁度の目が出れば止まることができます。

< TUTORIAL_05 >
最初のノルマは、スター10個を集めてホームマスに止まることです。\nそれを達成すると、次のノルマとして「スターを集める」か「バトルで勝利してウィンを集める」のどちらかを選びます。

< TUTORIAL_06 >
ノルマ選択で迷った時はスターノルマがオススメです。\n新しいチャプター開始時にスターが獲得できるので、いずれは目標数を集めることができます。

< TUTORIAL_07 >
黄色はボーナスマスです。\nダイスを振って、出た目の数×現在のノルマレベル(上限：3)のスターを獲得できます。

< TUTORIAL_08 >
青はドロップマスです。\nダイスを振って、出た目の数×現在のノルマレベル(上限：5)のスターを失います。

< TUTORIAL_09 >
緑はドローマスです。\nデッキからカードを引きます。

< TUTORIAL_10 >
赤はエネミーマスです。\n強制的に敵とのバトルになります。\n勝利するとスターとウィンを獲得できます。

< TUTORIAL_11 >
紫はワープマスです。\n別のワープマスへ移動します。

< TUTORIAL_12 >
その他にもマスがありますが、ぜひプレイしながら効果を確認してみてください！

< TUTORIAL_13 >
移動中に他のプレイヤーに遭遇した時は、そのプレイヤーにバトルを挑むことができます！

< TUTORIAL_14 >
ゲーム画面の四隅にあるキャラクターの顔にマウスカーソルをあてると、そのキャラクターの能力を確認できます。\nバトルを左右する重要な要素です。

< TUTORIAL_15 >
バトルでは、仕掛けた側が先攻で、攻撃ダイスを振ります。\n攻撃力補正があった場合は、出た目に補正値が付与されます。

< TUTORIAL_16 >
仕掛けられた側がDEFEND(防御)を選ぶと、ダイスを振り、被ダメージを最小で1に減らすことができます。

< TUTORIAL_17 >
EVADE(回避)を選ぶと、相手の攻撃ダイスよりも大きい目を出すことで、被ダメージを0にできます。\n失敗するとフルダメージを受けます。

< TUTORIAL_18 >
先攻の攻撃に耐えると、後攻が攻撃ダイスを振ります。

< TUTORIAL_19 >
HPが0になったプレイヤーはダウンします。\n勝ったプレイヤーは、負けたプレイヤーの所持スター半分と2ウィンを獲得します。

< TUTORIAL_20 >
次にカードについて。\n各プレイヤーは10枚ずつカードを選びます。\nさらにそこに空白のハイパーカード8枚を加えられ、シャッフルした上でセンターデッキとします。

< TUTORIAL_21 >
カード選びのポイントは、自分の選らんだユニットが有利になる効果を持つカードを見極めることです。

< TUTORIAL_22 >
各ユニットは固有のハイパーカードを持っています。\n空白のハイパーカードをデッキから引くと、そのユニットの固有ハイパーカードへ変化します。\nハイパーカードは強力な効果を持つものが多いので、どんどん使ってみてください！

< TUTORIAL_23 >
各カードの左側には1～5のレベルが表示されており、カードとプレイヤーのレベルが同数以上でないとそのカードは使えません。\nまた、カードの右側にはそのカードを使うの支払うコスト(スター)が表示されています。

< TUTORIAL_24 >
ゲームフィールドの多くには、\n「フィールド効果」があり、\n一定間隔で効果が発動します。\n「Info」ボタンにカーソルを合わせると、フィールドイベントなどの情報が確認できます。

< TUTORIAL_25 >
例えば、「チャリティ」というフィールド効果は4チャプターごとに発動し、各プレイヤーはデッキからカードを1枚引くことができます。

< TUTORIAL_26 >
対戦モードで誰かがノルマレベルが4に達すると、フィールド上にボスが出現し、エネミーマスがボスマスに変化します。\nボスは強力ですので、できるだけバトルを避けるのが得策です。

< TUTORIAL_27 >
対戦終了後、報酬としてスターを獲得できます。\nこのスターを使って、ゲーム内ショップにてカードなどのアイテムを入手できます。

< TUTORIAL_28 >
オンライン対戦、もしくはCO-OPモードの場合、報酬にオレンジが追加されます。\nオレンジは、ゲーム内ショップで様々なアクセサリーと交換できます。

< TUTORIAL_29 >
イベント開催中は、メインメニュー画面にあるイベントバナーをクリックすることで、イベントの詳細を確認できます。

< TUTORIAL_30 >
チュートリアルはこれで終了です。\nぜひゲームをお楽しみください！\nただし、まりーぽっぽにはくれぐれもご注意を…。
