< TUTORIAL_00 >
欢迎来到100%鲜橙汁！我们将讲解一些基础知识。\n当然，你也可以选择直接跳过。

< TUTORIAL_01 >
这是一款四人桌游，\n通常情况下胜利条件是将“目标”升至6级。

< TUTORIAL_02 >
你能通过投掷骰子在场地上移动，\n踩到不同颜色的格子会产生不同的效果。

< TUTORIAL_03 >
这是HOME格。\n你从这里开始游戏，你需要踩在上面才能升级\n或取得游戏的胜利。

< TUTORIAL_04 >
每名玩家都有自己的HOME格，写着自己的编号并能够主动停留在上面。\n但你也可以通过合适的骰子点数来踩在其他玩家的HOME格上。

< TUTORIAL_05 >
你的第一个目标是收集10个用于升级的\nSTAR，然后踩到HOME格上。\n之后你能选择下一个升级条件是收集STAR，\n还是通过赢得战斗来获得WIN。

< TUTORIAL_06 >
顺带一提，如果你不确定该选择哪个的话，\n推荐你选择STAR目标。\n因为你在每轮开始时都会获得更多的\nSTAR，永远都不会被卡住。

< TUTORIAL_07 >
黄色的格子是奖励格，使你可以通过投掷骰子\n获得STAR。\n你会获得你当前等级（最大倍率为3）×投掷\n点数的STAR。

< TUTORIAL_08 >
蓝色的格子是掉落格，踩到时会失去STAR。\n你会失去你当前等级（最大倍率为5）×投掷\n点数的STAR。

< TUTORIAL_09 >
绿色的格子是抽卡格，使你能够抽取一张卡。

< TUTORIAL_10 >
红色的格子是遭遇格，会强制使你与野怪\n进行战斗。\n你如果赢了，就会获得WIN和STAR。

< TUTORIAL_11 >
紫色的格子是传送格，它会把你带到随机的\n另一个拥有出口的传送格。

< TUTORIAL_12 >
还有一些更高级的格子……\n你能在游戏中搞清楚它们的用处！

< TUTORIAL_13 >
如果你在移动时遇到了另一名角色，\n你就能够对其发出挑战！

< TUTORIAL_14 >
把鼠标悬停在画面四角的角色头像上可以查看\n其他角色的能力值。\n能力值会影响角色在战斗中的表现。

< TUTORIAL_15 >
在战斗中，首先由先攻方投掷骰子进行攻击，\n拥有的ATK值会被加到骰子点数上。

< TUTORIAL_16 >
后攻方可以选择防御，会根据投掷的点数\n降低伤害，最低为1点。

< TUTORIAL_17 >
如果选择了闪避，后攻方必须投掷比进攻点数\n更高的数值才能消除伤害。\n否则会受到完整的伤害。

< TUTORIAL_18 >
如果能在第一次攻击中活下来，\n后攻方就能得到反击的机会！

< TUTORIAL_19 >
在一方的HP降至0时就会被击倒，\n胜者能得到2点WIN与败者手中50%的星星！

< TUTORIAL_20 >
接下来让我们来说说卡片！\n每名玩家可以选择10张卡片带进游戏中，\n它们会与8张空白HYPER卡一起在洗牌后\n放入卡堆。

< TUTORIAL_21 >
你需要挑选相比对于敌人，对你自己的角色\n更有帮助的卡片。

< TUTORIAL_22 >
每名角色都拥有独特的HYPER卡。\n如果在卡堆中抽到了空白HYPER卡，\n它会变为角色自己的HYPER卡。\n它们一般都会拥有非常强大的效果！

< TUTORIAL_23 >
需要注意的是，卡片拥有从1至5的等级，\n会被标记在卡片左侧，你需要到达\n相应的等级才能使用卡片。\n大多数卡片还需要消耗星星才能使用\n（被标记在右侧）。

< TUTORIAL_24 >
很多地图都拥有“场地效果”，它会以一定的\n间隔自动发动。\n把鼠标悬停在画面右侧的INFO按钮上可以\n查看场地效果与其它信息。

< TUTORIAL_25 >
举个例子，场地效果“恩惠”每5轮都会使\n所有玩家从卡堆中抽取一张卡片。

< TUTORIAL_26 >
在对战模式中，在任意一名玩家的等级\n到达4级时，一名强大的BOSS会出现在场上，将所有遭遇格替换为BOSS格。请小心移动！

< TUTORIAL_27 >
每场游戏结束后你会得到一些星星，\n你可以用它们商店中购买新的卡片和其它东西。

< TUTORIAL_28 >
在进行在线游戏或合作模式时，你同时会获得\n一些橙子，可以用来在商店中解锁各种装扮。

< TUTORIAL_29 >
如果有活动正在进行，你可以点击主菜单的\n活动横幅来查看更多信息。

< TUTORIAL_30 >
祝你玩得愉快……另外请小心玛丽啵噗！
