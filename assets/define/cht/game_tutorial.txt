< TUTORIAL_00 >
歡迎來到100%鮮橙汁！我們將講解一些基礎知識。\n當然，你也可以選擇直接跳過。

< TUTORIAL_01 >
這是一款四人桌遊，\n通常情況下勝利條件是將“目標”升至6級。

< TUTORIAL_02 >
你能通過投擲骰子在場地上移動，\n踩到不同顏色的格子會產生不同的效果。

< TUTORIAL_03 >
這是HOME格。\n你從這裡開始遊戲，你需要踩在上面才能升級\n或取得遊戲的勝利。

< TUTORIAL_04 >
每名玩家都有自己的HOME格，寫著自己的編號並能夠主動停留在上面。\n但你也可以通過合適的骰子點數來踩在其他玩家的HOME格上。

< TUTORIAL_05 >
你的第一個目標是收集10個用於升級的\nSTAR，然後踩到HOME格上。\n之後你能選擇下一個升級條件是收集STAR，\n還是通過贏得戰鬥來獲得WIN。

< TUTORIAL_06 >
順帶一提，如果你不確定該選擇哪個的話，\n推薦你選擇STAR目標。\n因為你在每輪開始時都會獲得更多的\nSTAR，永遠都不會被卡住。

< TUTORIAL_07 >
黃色的格子是獎勵格，使你可以通過投擲骰子\n獲得STAR。\n你會獲得你當前等級（最大倍率為3）×投擲\n點數的STAR。

< TUTORIAL_08 >
藍色的格子是掉落格，踩到時會失去STAR。\n你會失去你當前等級（最大倍率為5）×投擲\n點數的STAR。

< TUTORIAL_09 >
綠色的格子是抽卡格，使你能夠抽取一張卡。

< TUTORIAL_10 >
紅色的格子是遭遇格，會強制使你與野怪\n進行戰鬥。\n你如果贏了，就會獲得WIN和STAR。

< TUTORIAL_11 >
紫色的格子是傳送格，它會把你帶到隨機的\n另一個擁有出口的傳送格。

< TUTORIAL_12 >
還有一些更高級的格子……\n你能在遊戲中搞清楚它們的用處！

< TUTORIAL_13 >
如果你在移動時遇到了另一名角色，\n你就能夠對其發出挑戰！

< TUTORIAL_14 >
把鼠標懸停在畫面四角的角色頭像上可以查看\n其他角色的能力值。\n能力值會影響角色在戰鬥中的表現。

< TUTORIAL_15 >
在戰鬥中，首先由先攻方投擲骰子進行攻擊，\n擁有的ATK值會被加到骰子點數上。

< TUTORIAL_16 >
後攻方可以選擇防禦，會根據投擲的點數\n降低傷害，最低為1點。

< TUTORIAL_17 >
如果選擇了閃避，後攻方必須投擲比進攻點數\n更高的數值才能消除傷害。\n否則會受到完整的傷害。

< TUTORIAL_18 >
如果能在第一次攻擊中活下來，\n後攻方就能得到反擊的機會！

< TUTORIAL_19 >
在一方的HP降至0時就會被擊倒，\n勝者能得到2點WIN與敗者手中50%的星星！

< TUTORIAL_20 >
接下來讓我們來說說卡片！\n每名玩家可以選擇10張卡片帶進遊戲中，\n它們會與8張空白HYPER卡一起在洗牌後\n放入卡堆。

< TUTORIAL_21 >
你需要挑選相比對於敵人，對你自己的角色\n更有幫助的卡片。

< TUTORIAL_22 >
每名角色都擁有獨特的HYPER卡。\n如果在卡堆中抽到了空白HYPER卡，\n它會變為角色自己的HYPER卡。\n它們一般都會擁有非常強大的效果！

< TUTORIAL_23 >
需要注意的是，卡片擁有從1至5的等級，\n會被標記在卡片左側，你需要到達\n相應的等級才能使用卡片。\n大多數卡片還需要消耗星星才能使用\n（被標記在右側）。

< TUTORIAL_24 >
很多地圖都擁有“場地效果”，它會以一定的\n間隔自動發動。\n把鼠標懸停在畫面右側的INFO按鈕上可以\n查看場地效果與其它信息。

< TUTORIAL_25 >
舉個例子，場地效果“恩惠”每5輪都會使\n所有玩家從卡堆中抽取一張卡片。

< TUTORIAL_26 >
在對戰模式中，在任意一名玩家的等級\n到達4級時，一名強大的BOSS會出現在場上，將所有遭遇格替換為BOSS格。請小心移動！

< TUTORIAL_27 >
每場遊戲結束後你會得到一些星星，\n你可以用它們商店中購買新的卡片和其它東西。

< TUTORIAL_28 >
在進行在線遊戲或合作模式時，你同時會獲得\n一些橙子，可以用來在商店中解鎖各種裝扮。

< TUTORIAL_29 >
如果有活動正在進行，你可以點擊主菜單的\n活動橫幅來查看更多信息。

< TUTORIAL_30 >
祝你玩得愉快……另外請小心瑪麗啵噗！
